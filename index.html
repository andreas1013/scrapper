🚀 Cloudways deployment steps

  1. Create a new PHP application on Cloudways (or use a blank HTML app)
  2. Download the ZIP from this button
  3. Upload ZIP via SFTP or Cloudways File Manager into public_html/
  4. Extract the ZIP — your site structure will be in place
  5. The .htaccess handles all 301 redirects automatically
  6. Review each page and replace placeholder content with your real content
